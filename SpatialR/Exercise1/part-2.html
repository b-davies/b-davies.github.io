2.2 Plotting spatial data

Next weâ€™ll use plot and st_geometry to plot the borders.

While we all probably have a sense of where southern Africa is, itâ€™s usually a good idea to include some information about the geography. Weâ€™ll learn more about making maps later, but for now we can just include the axes to give us latitude and longitude.

If we want to see the information in the dataset, we could just use the plot function by itself.
